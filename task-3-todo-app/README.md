# Level 2: Functional To-Do List App

A professional, responsive Task Management application built as part of my **Codveda Internship**. This project focuses on CRUD operations, data persistence, and modern UI/UX principles.

## 🚀 Key Features
- **Persistence:** Uses `localStorage` to save tasks, ensuring data remains after page refreshes.
- **Categorization:** Automatically groups tasks into "To Do" and "Completed" sections.
- **Intuitive UI:** Integrated checkboxes with full-row clickability using HTML labels for better accessibility.
- **Responsive Design:** Clean, centered layout built with custom Vanilla CSS.

## 🛠️ Technical Implementation
- **State-Driven UI:** The interface is a reflection of the JavaScript data array. Any change to the data triggers a re-render.
- **DOM Manipulation:** Efficient use of `innerHTML` and `createElement` to dynamically update the view.
- **Boolean Logic:** Implemented the NOT operator (`!`) for a seamless task toggle system.

## 📂 Project Structure
- `index.html`: Semantic HTML5 structure.
- `style.css`: Custom CSS for layout, Flexbox alignment, and state-based styling.
- `script.js`: Core logic for data handling and UI updates.

## 📝 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
